# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- `parse()` — parse CSS color strings (hex, rgb, hsl, named colors)
- `toHex()` — convert Color to hex string
- `toRgb()` — convert Color to rgb/rgba string
- `toHsl()` — convert Color to hsl/hsla string
- `isValid()` — check if a string is a valid CSS color
- 30+ named CSS colors supported
